# Quark App

基于本工程，您可以构建前端应用。

可以用下面命令生成本工程模版！

```bash
npm create quarkc@latest
```

文档见：https://quark-ecosystem.github.io/quarkc-docs/#/

无框架前端应用模版，底层基于 Web components。

> 本工程用于脱离 React、Vue 等技术栈来构建一个独立的 Web 应用，[技术文档](https://github.com/hellof2e/quark)。

- [x] 支持路由
- [x] 支持数据绑定
- [x] 没有虚拟 DOM
- [x] 支持 TSX / JSX
- [x] 支持将 Markdown 渲染成 html
- [x] 浏览器原生，没有依赖
- [x] 开发环境支持热更新


## 如何使用

```
npm install
npm run dev
```

## 打包产物

```
npm run build
```
